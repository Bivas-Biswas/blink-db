\doxysubsection{Detailed Description}
Command-\/line interface for interacting with the LRU cache database. 

This program provides a simple CLI to interact with an LRUCache-\/based key-\/value store. Supported commands\+:
\begin{DoxyItemize}
\item SET $<$key$>$ 
\end{DoxyItemize}

\+: Stores the key-\/value pair.
\begin{DoxyItemize}
\item GET $<$key$>$\+: Retrieves the value for a given key.
\item DEL $<$key$>$\+: Deletes a key from the cache.
\item EXIT\+: Terminates the program. 
\end{DoxyItemize}
